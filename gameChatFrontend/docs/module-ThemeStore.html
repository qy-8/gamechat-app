Module: ThemeStore

Pinia Store，用于管理应用程序的亮/暗主题模式。 该 Store 会自动同步主题状态到 `` 元素的 class 列表，并支持持久化。

Methods

(inner) initTheme() → {void}

初始化主题模式。 如果 `localStorage` 中没有 'theme' 的值，则会检查用户的系统颜色偏好（ prefers-color-scheme: dark ）来设置初始主题。 如果 `persist: true` 已启用，此函数通常会在 Pinia 恢复状态后运行。
